Terina App

Terina App is an educational biomedical engineering project for managing and reviewing retinal imaging data related to Diabetic Retinopathy (DR) and Diabetic Macular Edema (DME).

The system provides a MySQL database, backend logic, an internal AppAPI communication layer, and a Tkinter graphical interface for managing patients, medical profiles, scan sessions, and OCT and Fundus analysis results.

The application uses deterministic rule-based clinical analysis to calculate risk levels, summaries, and recommendations. Patient information is synthetic, and the project is intended only as a clinical decision-support infrastructure model for a Health Information Systems course.

Dataset

This project uses the public OCT and Eye Fundus Dataset provided by the Translational Visual Health Laboratory:

[OCT and Eye Fundus Dataset](https://github.com/Traslational-Visual-Health-Laboratory/OCT-AND-EYE-FUNDUS-DATASET)

The dataset is used for educational and research purposes. All image data and medical labels belong to the original dataset authors.

This project is not intended for real clinical diagnosis or treatment decisions.
