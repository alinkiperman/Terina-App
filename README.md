Terina App

Terina App is a biomedical engineering project focused on the management and analysis of retinal imaging data related to Diabetic Retinopathy (DR) and Diabetic Macular Edema (DME).

The system provides database, backend, and communication layer infrastructure for managing patients, medical profiles, scan sessions, retinal imaging records, and analysis results.


Dataset

This project uses the public OCT and Eye Fundus Dataset provided by the Translational Visual Health Laboratory:

https://github.com/Traslational-Visual-Health-Laboratory/OCT-AND-EYE-FUNDUS-DATASET

The dataset is used for educational and research purposes. All image data and medical labels belong to the original dataset authors.
