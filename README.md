# Terina App

Terina App is an educational biomedical engineering project for managing and reviewing medical imaging data.

The system provides a MySQL database, backend logic, an internal AppAPI communication layer, and a user interface for managing patients, medical profiles, scan sessions, and analysis results.

Patient information is synthetic, and the project is intended solely as a clinical decision-support infrastructure prototype developed as part of a Health Information Systems course.

## Dataset

[OCT and Eye Fundus Dataset](https://github.com/Traslational-Visual-Health-Laboratory/OCT-AND-EYE-FUNDUS-DATASET)

The dataset is used for educational and research purposes. All imaging data and medical labels belong to the original dataset authors.

This project is not intended for real-world clinical diagnosis or treatment decisions.
